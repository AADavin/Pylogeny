Class Hierarchy

  • pylogeny.pll.dataModel: Encapsulating a phylogenetic tree (as topology) + corresponding alignment into a libpll-associated data structure.
  • pylogeny.landscapeWriter.landscapeParser: Encapsulates the construction of a landscape object from a pickle file.
  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • pylogeny.newick.ParsingError
        • pylogeny.model.PhyloModelError
        • pylogeny.rearrangement.RearrangementError
    • pylogeny.model.DiscreteStateModel: Initialize a discrete state model for phylogenetic data.
    • pylogeny.executable.aTemporaryDirectory: A class intended to be used as a context manager that allows Python to run in another directory temporarily.
    • pylogeny.alignment.alignment: Wrap a biological sequence alignment to enable functionality necessary for phylogenetic inference.
      • pylogeny.alignment.phylipFriendlyAlignment: An alignment object that renames all comprising taxa in order to be able to be written as a Phylip file.
    • pylogeny.tree.bipartition: A tree bipartition.
    • pylogeny.newick.branch: Newick branch.
    • pylogeny.database.database: Allow interfacing with a SQL/sqlite database.
      • pylogeny.database.SQLDatabase: Database object to allow reading from a MySQL database.
      • pylogeny.database.SQLiteDatabase
    • pylogeny.executable.executable: An interface for the instantation and running of a single instance for a given application.
      • pylogeny.executable.consel: Denotes a single run of the CONSEL workflow in order to acquire a confidence interval and perform an AU test on a set of trees.
      • pylogeny.executable.fasttree: Denotes a single run of the FastTree executable in order to acquire an approximate maximum likelihood tree for the input alignment.
      • pylogeny.executable.raxml: Denotes a single run of the RAxML executable.
      • pylogeny.executable.treepuzzle: Wrap TREE-PUZZLE in order to create an intermediate file for CONSEL to read and assign confidence to a set of trees.
    • pylogeny.landscape.graph: Define an empty graph object.
      • pylogeny.landscape.landscape: Defines an entire phylogenetic tree space.
        • pylogeny.database.DatabaseLandscape: Abstract the landscape to one comprising a landscape.
          • pylogeny.database.SQLExhaustiveLandscape
        • pylogeny.database.SQLiteLandscape: Allow random access of all landscape data from an sqlite file found on the hard disk.
    • pylogeny.heuristic.heuristic: A base interface for a heuristic that explores a state graph.
      • pylogeny.heuristic.phylogeneticLinearHeuristic: A base class for a heuristic that works on a phylogenetic landscape and only possesses a single path (of search).
        • pylogeny.heuristic.RAxMLIdentify: RAxML-driven landscape evaluation of intermediate checkpoint trees output from the RAxML executable.
        • pylogeny.heuristic.likelihoodGreedy: Greedy (hill-climbing) landscape exploration by comparsion of likelihood.
        • pylogeny.heuristic.parsimonyGreedy: Greedy (hill-climbing) landscape exploration by comparsion of parsimony.
        • pylogeny.heuristic.smoothGreedy: Parsimony-driven greedy landscape exploration by comparsion of likelihoods.
    • pylogeny.landscapeWriter.landscapeWriter: Encapsulate the writing of a landscape to a file format.
      • pylogeny.JSONWriter.JSONWriter
    • pylogeny.newick.node: Newick node.
    • pylogeny.tree.tree: Defines a single (phylogenetic) tree by newick string; can possess other metadata.
    • pylogeny.tree.treeSet: Represents an ordered, unorganized collection of trees that do not necessarily comprise a combinatorial space.
      • pylogeny.landscape.landscape: Defines an entire phylogenetic tree space.
        • pylogeny.database.DatabaseLandscape: Abstract the landscape to one comprising a landscape.
          • pylogeny.database.SQLExhaustiveLandscape
        • pylogeny.database.SQLiteLandscape: Allow random access of all landscape data from an sqlite file found on the hard disk.
    • type: type(object) -> the object's type type(name, bases, dict) -> a new type
      • abc.ABCMeta: Metaclass for defining Abstract Base Classes (ABCs).
    • pylogeny.landscape.vertex: Encapsulate a single vertex in the landscape and add convenient functionality to alias parent landscape functions.
  • pylogeny.newick.parser: Parsing object for Newick strings.
  • pylogeny.pll.partitionModel: A partition model intended for libpll.
  • pylogeny.parsimony.profile_set: Hold a set of site_profile profiles for an entire alignment.
  • pylogeny.rearrangement.rearrangement: Encapsulates a single rearrangement move of type SPR, NNI, ...
  • pylogeny.parsimony.site_profile: Consolidate the single-column alignment at a region into a set of components on the basis of similarity alone.
  • pylogeny.rearrangement.topology: Encapsulate a tree topology, wrapping the newick tree structure.
