Package pylogeny

source code

Pylogeny is a Python library and code framework for phylogenetic tree reconstruction and scoring.

Allows one to perform the following tasks: (1) Generate and manage phylogenetic tree landscapes. (2) Build and rearrange phylogenetic trees using preset operators such as NNI, SPR, and TBR. (3) Score phylogenetic trees by Log-likelihood and Parsimony.

Dependencies: Pandas, P4 Phylogenetic Library. Suggested: FastTree, RAxML, PytBEAGLEhon.

Submodules [hide private]
  • pylogeny.JSONWriter: Serialize a phylogenetic landscape into a JSON object.
  • pylogeny.alignment: Handle input biological sequence alignment files for the purposes of phylogenetic inference.
  • pylogeny.database: Connect, access, + manipulate external tree data from a remote SQL server or from a sqlite file.
  • pylogeny.executable: Defines an interface to manage interfacing with the system for respective application calls and implements multiple of these for executables such as FastTree and RAxML.
  • pylogeny.heuristic: Define the interface for a heuristic in order to implement any manner of heuristic for a combinatorial problem that can be abstracted into a state graph.
  • pylogeny.landscape: Encapsulate a phylogenetic tree space.
  • pylogeny.landscapeWriter: Serialize a phylogenetic landscape into an archive made up of three components: a flatfile containing all tree IDs and respective scores, the alignment file written as a FASTA file, and a representation of the graph in the format.
  • pylogeny.model: Phylogenetic tree scoring models; intended to be coupled with the use of pytbeaglehon (BEAGLE) high-performance library.
  • pylogeny.newick: Newick string parsing and object interaction.
  • pylogeny.parsimony: Toolkit for performance of Parsimonious Criterion (Parsimony) methods of optimization of a phylogenetic topology with a particular set of data.
  • pylogeny.pll: C Extension to wrap libpll library.
  • pylogeny.rearrangement: Phylogenetic tree structure encapsulation; allow rearrangement of said structure.
  • pylogeny.scoring: Phylogenetic tree scoring.
  • pylogeny.tree: Container definition for (phylogenetic) bifurcating or multifurcating trees defined using Newick strings.
