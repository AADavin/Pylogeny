pylogeny.scoring module¶

Phylogenetic tree scoring.

pylogeny.scoring.beaglegetLogLikelihood(tree, alignment)[source]¶

Acquire log-likelihood via C++ library BEAGLE via use of pybeaglethon wrapper library. Parameters: newick.tree object and alignment object. Currently uses HKY85 model.

pylogeny.scoring.getLogLikelihood(tree, alignment)[source]¶

Acquire log-likelihood via C library libpll. Parameters: newick.tree object and alignment object.

pylogeny.scoring.getLogLikelihoodForTopology(topo, alignment)[source]¶

Acquire log-likelihood via C library libpll. Parameters: rearrangement.topology object and alignment object.

pylogeny.scoring.getParsimony(newick, alignment)[source]¶

Acquire parsimony via a C++ implementation. Parameters: newick string and alignment object.

pylogeny.scoring.getParsimonyForTopology(topology, alignment)[source]¶

Acquire parsimony via a C++ implementation. Parameters: rearrangement.topology and alignment object.

pylogeny.scoring.getParsimonyFromProfiles(newick, profiles)[source]¶

Acquire parsimony via a C++ implementation. Parameters: newick string and parsimony.profile_set object.

pylogeny.scoring.getParsimonyFromProfilesForTopology(topology, profiles)[source]¶

Acquire parsimony via a C++ implementation. Parameters: rearrangement.topology and parsimony.profile_set object.
